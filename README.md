## English Description

A fully functional browser-based chess game developed using HTML, CSS, and vanilla JavaScript, without any third-party libraries for game logic.

### Key Features:
* **Game Engine:** Complete implementation of classical chess rules (castling, pawn promotion, check/mate/stalemate detection).
* **Artificial Intelligence:** Advanced AI opponent powered by the **Minimax algorithm** with alpha-beta pruning. Three difficulty levels: Easy, Medium, and Pro.
* **Training Mode:** Built-in chess puzzles (Mate in 1, 2, or 3 moves) with automatic solution verification.
* **User Interface:** * Visual move hints.
    * Undo move functionality (available for Easy/Medium modes).
    * Customizable themes (Classic, Neon, Ocean, Wood).
    * Switchable landscape backgrounds.
* **Audio:** Integrated sound effects for moves, captures, and victory.

## Описание проекта (Русский)

Полнофункциональная браузерная игра в шахматы, разработанная с использованием HTML, CSS и чистого JavaScript без использования сторонних библиотек для игровой логики.

### Основные возможности:
* **Игровой движок:** Полная реализация правил классических шахмат (рокировка, превращение пешки, определение шаха, мата и пата).
* **Искусственный интеллект:** Продвинутый ИИ-соперник на базе алгоритма **минимакса (Minimax)** с альфа-бета отсечением. Три уровня сложности: Легкий, Средний и Профи.
* **Режим тренировки:** Встроенные шахматные задачи (Мат в 1, 2 и 3 хода) с автоматической проверкой решения.
* **Интерфейс:** * Подсветка доступных ходов.
    * Функция отмены хода (доступна на легком и среднем уровнях).
    * 4 стиля оформления (Классика, Неон, Океан, Дерево).
    * Возможность смены фоновых пейзажей.
* **Звук:** Интегрированные звуковые эффекты для ходов, взятия фигур и завершения партии.
